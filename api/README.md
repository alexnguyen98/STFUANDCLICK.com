## @stfuandclick/api

Nodejs express server with Typeorm as ORM and Postgres database

```sh
# project structure
api/src/
    ├── api
    ├── config
    ├── entity
    ├── middlewares
    └── types
    - app
```

### Setup

### Setup

```bash
# install dependencies
yarn install

# Create a copy of .env-example as .env and fill the needed configuration

# running dev mode
yarn dev

# running for production
yarn start
```
